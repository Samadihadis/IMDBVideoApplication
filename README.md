<h2>IMDBVideoApplication</h2>

<h4>Simple IMDB Video List and Detail</h4>

Feature's:
- Constraintlayout
- RecyclerView
- Navigation Component(action, argument, backstack management)
- Glide
- OkHttp
- Gson Convertor
- Animation
- Adapter and ViewHolder
- Util Class
- Loading State
- IMDB API = https://api.themoviedb.org/3/movie/popular
